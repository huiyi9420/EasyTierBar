## 项目概述

EasyTierBar 是一个 macOS 菜单栏工具，为 [EasyTier](https://github.com/EasyTier/EasyTier) VPN 提供图形化操作界面。基于 Swift + Cocoa 原生开发，面向 Apple Silicon (aarch64)。

## 本次完成的工作

### 1. 核心架构重构：easytier-cli service → launchd 直接管理

**问题**: `easytier-cli service install/start/stop`（基于 service-manager-rs）在 macOS 上存在 Launchd 兼容性问题（GitHub issue #1599），导致服务停止时报错 `Could not find service "easytier" in domain for system`。

**解决方案**: 放弃 `easytier-cli service`，改为直接管理 launchd 服务：

| 操作 | 旧方案 | 新方案 |
|------|--------|--------|
| 启动 | `easytier-cli service start` | 写 launchd plist + `launchctl load` |
| 停止 | `easytier-cli service stop` | `launchctl unload` + `pkill` 兜底 |
| 状态 | `easytier-cli service status` | `pgrep -x easytier-core` |
| 崩溃恢复 | 无 | launchd `KeepAlive: true` 自动重启 |

**修改文件**: `EasyTierBar/ServiceManager.swift`

### 2. UI 体验修复

- **节点列表状态提示**: 服务停止时显示"(服务未启动)"而非"(获取节点列表失败)"

## 当前架构

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  EasyTierBar │────→│ ServiceManager   │────→│ launchd         │
│  (菜单栏 UI) │     │ (Swift, 管理层)  │     │ (系统服务管理器) │
└─────────────┘     └──────────────────┘     └────────┬────────┘
       │                    │                          │
       │                    │ sudo (AppleScript)        │
       ▼                    ▼                          ▼
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│ ConfigManager│     │ /Library/Launch  │     │ easytier-core   │
│ (UserDefaults)│     │ Daemons/        │     │ (VPN 后台进程)   │
└─────────────┘     │ com.easytier.    │     └─────────────────┘
                    │ core.plist       │
                    └──────────────────┘
```

**服务管理流程**:
1. 启动: 生成 launchd plist → `cp` 到 `/Library/LaunchDaemons/` → `launchctl load`
2. 停止: `launchctl unload` → 删除 plist → `pkill` 兜底
3. 状态: `pgrep -x easytier-core` 检测进程
4. 权限: AppleScript `do shell script with administrator privileges` 弹出系统密码框

**数据流**:
- 配置存储: `UserDefaults` (配置名称、URL、选中项)
- 运行日志: `/tmp/easytier-core.log`
- 调试日志: `os_log` (subsystem: `com.easytier.bar`)

## 文件结构

```

```

## 关键技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 服务管理 | launchd plist | macOS 原生，可靠；easytier-cli service 在 macOS 有兼容性问题 |
| 权限提升 | AppleScript `do shell script` | macOS 标准方式，用户友好（系统密码框） |
| 进程检测 | `pgrep -x` | 简单可靠，无需解析输出 |
| 日志系统 | `os_log` | 系统级日志，可通过 `log stream` 查看 |
| 崩溃恢复 | `KeepAlive: true` | VPN 服务应自动恢复 |

## 已知限制

1. 仅支持 macOS aarch64 (Apple Silicon)
2. 服务管理需要管理员密码（TUN 设备创建需要 root）
3. `easytier-cli peer list` 查询可能因核心版本差异返回不同字段
4. 首次安装需手动绕过 Gatekeeper

## 用户体验

安装 → 右键打开 → 添加配置 → 一键连接。日常使用只需点击菜单栏图标切换连接状态。

```
启动服务: 点击「启动 EasyTier」→ 输入密码 → 等待 1-2 秒 → 图标变为已连接
停止服务: 点击「停止 EasyTier」→ 输入密码 → 图标变为未连接
查看节点: 点击「已连接节点」→ 展开节点列表
```
