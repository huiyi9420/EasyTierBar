# Plan 01: GitHub Actions CI/CD Workflow

## Objective

创建 `.github/workflows/build.yml`，实现完全自动化的构建流水线：定时检测上游 EasyTier 新版本 → 下载二进制 → 编译 Swift → 打包 app → 发布 Release。

## Tasks

### Task 1: 创建 GitHub Actions workflow

<read_first>
- Makefile — 了解编译命令
- EasyTierBar/Info.plist — 了解 app bundle 配置
</read_first>

<action>
创建文件 `.github/workflows/build.yml`：

```yaml
name: Build EasyTierBar

on:
  schedule:
    - cron: '0 */6 * * *'
  workflow_dispatch:
    inputs:
      easytier_tag:
        description: 'EasyTier release tag (leave empty for latest)'
        required: false
        type: string

permissions:
  contents: write

jobs:
  check-and-build:
    runs-on: macos-15
    outputs:
      built_tag: ${{ steps.check.outputs.tag }}
    steps:
      - name: Checkout
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Check for new EasyTier release
        id: check
        run: |
          if [ -n "${{ inputs.easytier_tag }}" ]; then
            TAG="${{ inputs.easytier_tag }}"
          else
            TAG=$(curl -sf https://api.github.com/repos/EasyTier/EasyTier/releases/latest | jq -r '.tag_name')
          fi

          if [ -z "$TAG" ] || [ "$TAG" = "null" ]; then
            echo "No EasyTier release found"
            echo "skip=true" >> $GITHUB_OUTPUT
            exit 0
          fi

          # Check if we already have a release for this tag
          EXISTING=$(gh release view "easytierbar-${TAG}" 2>/dev/null || true)
          if [ -n "$EXISTING" ]; then
            echo "Release for ${TAG} already exists — skipping"
            echo "skip=true" >> $GITHUB_OUTPUT
            exit 0
          fi

          echo "tag=$TAG" >> $GITHUB_OUTPUT
          echo "skip=false" >> $GITHUB_OUTPUT
          echo "Building for EasyTier ${TAG}"
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}

      - name: Download EasyTier binaries
        if: steps.check.outputs.skip == 'false'
        run: |
          TAG="${{ steps.check.outputs.tag }}"
          URL="https://github.com/EasyTier/EasyTier/releases/download/${TAG}/easytier-aarch64-apple-darwin.zip"
          echo "Downloading ${URL}"
          curl -L -o easytier.zip "${URL}"
          unzip easytier.zip
          # Find easytier-cli and easytier-core in extracted directory
          find . -name "easytier-cli" -exec mv {} . \;
          find . -name "easytier-core" -exec mv {} . \;
          chmod +x easytier-cli easytier-core
          ls -la easytier-*

      - name: Build EasyTierBar
        if: steps.check.outputs.skip == 'false'
        run: |
          swiftc -o EasyTierBar \
            EasyTierBar/main.swift \
            EasyTierBar/ConfigManager.swift \
            EasyTierBar/ServiceManager.swift \
            EasyTierBar/AppDelegate.swift \
            -framework Cocoa

      - name: Package app bundle
        if: steps.check.outputs.skip == 'false'
        run: |
          APP="EasyTierBar.app"
          mkdir -p "${APP}/Contents/MacOS"
          mkdir -p "${APP}/Contents/Resources"
          mv EasyTierBar "${APP}/Contents/MacOS/"
          cp EasyTierBar/Info.plist "${APP}/Contents/"
          mv easytier-cli easytier-core "${APP}/Contents/Resources/"
          chmod +x "${APP}/Contents/Resources/"*
          echo "App bundle contents:"
          find "${APP}" -type f

      - name: Create release archive
        if: steps.check.outputs.skip == 'false'
        run: |
          zip -r EasyTierBar-aarch64-macos.zip EasyTierBar.app/

      - name: Create GitHub Release
        if: steps.check.outputs.skip == 'false'
        run: |
          TAG="${{ steps.check.outputs.tag }}"
          gh release create "easytierbar-${TAG}" \
            --title "EasyTierBar ${TAG}" \
            --notes "Auto-built EasyTierBar based on EasyTier ${TAG}

          ## Included
          - EasyTierBar.app (macOS aarch64 / Apple Silicon)
          - easytier-cli ${TAG}
          - easytier-core ${TAG}

          ## Installation
          1. Download and unzip
          2. Move EasyTierBar.app to /Applications
          3. First launch: right-click → Open (bypass Gatekeeper)
          4. Menu bar → add network config → connect" \
            EasyTierBar-aarch64-macos.zip
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
</action>
