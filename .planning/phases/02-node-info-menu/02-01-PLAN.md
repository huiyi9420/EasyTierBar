# Plan 01: Node Info Submenu — 节点信息子菜单

## Objective

在 AppDelegate 的主菜单中添加"已连接节点"二级子菜单，通过 `easytier-cli -o json peer list` 获取节点数据，解析 JSON 并显示每个节点的 7 个字段。菜单打开时自动刷新。

## Tasks

### Task 1: 在 ServiceManager 中添加 fetchPeerList 方法

<read_first>
- EasyTierBar/ServiceManager.swift — 当前文件
</read_first>

<action>
在 `ServiceManager` 类中添加 peer list 获取方法：

```swift
    // MARK: - Peer List (Async)

    struct Peer: Codable {
        let hostname: String
        let ipv4: String
        let cost: String
        let lat_ms: String
        let loss_rate: String
        let rx_bytes: String
        let tx_bytes: String
        let version: String
    }

    func fetchPeerList(completion: @escaping ([Peer]?) -> Void) {
        guard let path = cliPath else {
            completion(nil)
            return
        }

        DispatchQueue.global(qos: .userInitiated).async {
            let process = Process()
            process.executableURL = URL(fileURLWithPath: path)
            process.arguments = ["-o", "json", "peer", "list"]
            let pipe = Pipe()
            process.standardOutput = pipe
            process.standardError = FileHandle.nullDevice

            do {
                try process.run()
                process.waitUntilExit()
                let data = pipe.fileHandleForReading.readToEnd() ?? Data()
                let peers = try? JSONDecoder().decode([Peer].self, from: data)
                DispatchQueue.main.async {
                    completion(peers)
                }
            } catch {
                DispatchQueue.main.async {
                    completion(nil)
                }
            }
        }
    }
```
</action>

<acceptance_criteria>
- ServiceManager 有 `Peer` struct，包含 8 个字段（hostname, ipv4, cost, lat_ms, loss_rate, rx_bytes, tx_bytes, version）
- ServiceManager 有 `fetchPeerList(completion:)` 方法
- 使用 DispatchQueue.global 异步执行
- JSON 解码结果回到主线程
- cliPath 为 nil 或执行失败时返回 nil
</acceptance_criteria>

3. 添加 `updatePeerList()` 方法：
```swift
    private func updatePeerList() {
        ServiceManager.shared.fetchPeerList { [weak self] peers in
            guard let self = self else { return }
            self.peerMenu.removeAllItems()

            if let peers = peers, !peers.isEmpty {
                for peer in peers {
                    let title = "\(peer.hostname) • \(peer.ipv4) • \(peer.cost) • \(peer.lat_ms)ms • \(peer.loss_rate) • \(peer.rx_bytes)/\(peer.tx_bytes) • \(peer.version)"
                    let item = NSMenuItem(title: title, action: nil, keyEquivalent: "")
                    self.peerMenu.addItem(item)
                }
            } else {
                let title = peers == nil ? "(获取节点列表失败)" : "(无已连接节点)"
                let item = NSMenuItem(title: title, action: nil, keyEquivalent: "")
                item.isEnabled = false
                self.peerMenu.addItem(item)
            }
        }
    }
```

4. 在 `menuWillOpen()` 方法中添加 `updatePeerList()` 调用：
```swift
    @objc private func menuWillOpen() {
        ServiceManager.shared.checkStatus()
        updatePeerList()
    }
```

5. 在 `applicationDidFinishLaunching` 中添加初始节点列表：
```swift

        updatePeerList()
```
